Cohort Retention Analysis

Modeled cohorts by signup month, built retention matrices, and identified high‑value channels. Improved M3 retention by 6% after spend reallocation.

-- Example: new vs returning revenue by month
WITH rev AS (
  SELECT DATE_TRUNC(order_date, MONTH) AS m,
         user_id,
         SUM(amount) AS revenue,
         MIN(order_date) OVER (PARTITION BY user_id) AS first_order
  FROM analytics.orders
  GROUP BY 1,2
)
SELECT m,
  SUM(CASE WHEN DATE_TRUNC(first_order, MONTH)=m THEN revenue ELSE 0 END) AS new_rev,
  SUM(CASE WHEN DATE_TRUNC(first_order, MONTH)<>m THEN revenue ELSE 0 END) AS returning_rev
FROM rev
GROUP BY 1
ORDER BY 1;
